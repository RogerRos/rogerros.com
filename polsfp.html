POLS Final Project

This is an interactive exhibition of my POLS course final project. Each orbiting object represents a section of the project.

Drag to orbit the view. Scroll to zoom. Click any object to open its content. Press Escape or Back to return.

Reflection

My apologies. This is not binary. It is input rendered as single random bits for dramatic effect. Real binary would require the full 8-bit ASCII representation of each character, which would take eight times the space and would not be nearly as dramatic. I suppose we should blame Hollywood.
